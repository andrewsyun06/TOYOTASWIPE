# TOYOTASWIPE

ToyotaSwipe is a Flutter-based mobile application designed to help users explore and discover Toyota vehicles through an interface inspired by modern short-form social media platforms.

The application allows users to swipe through Toyota models, compare specifications, explore pricing information, and access immersive vehicle experiences through virtual tours. ToyotaSwipe aims to create an engaging car discovery platform while directing users toward Toyota's official resources for additional information.

---

## Features

### Vehicle Exploration
Browse Toyota vehicles through an intuitive swipe-based interface designed for fast and engaging discovery.

### Virtual Vehicle Tours
Access interactive virtual tours of supported Toyota models, allowing users to explore vehicles in a more immersive environment.

### Vehicle Specifications
View important vehicle information including:

- Pricing
- Miles per gallon (MPG)
- Available colors
- Official Toyota website links
- Model-specific details

### Compare Vehicles
Compare and contrast Toyota models to help users identify vehicles that best match their preferences.

## Installation

### Prerequisites

Install the following before running the project:

- Flutter SDK
- Android Studio or Visual Studio Code
- Android emulator or iOS simulator

### Run Locally

```bash
git clone <repository-url>
cd toyotaswipe
flutter pub get
flutter run
```
